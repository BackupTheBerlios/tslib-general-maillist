afunix <afunix at hing.rednet.ru> wrote:

> Hello
> 
> I've installed debian lenny on dell axim. 
> I've calibrated touchscreen with ts_calibrate. 
> When I'm using ts_test a can access every corner of my screen with stylus.
> But when I'm starting X, I can use only some part of it. When I move stylus to top-left corner of screen, mouse-pointer is in top-left corner too, right under stylus. But when I move stylus to bottom-left corner, pointer is somewhere in the middle of the screen.

This sounds like an issue with the old version of the Xorg driver in
Debian - the current version in lenny.

See http://bugs.debian.org/cgi-bin/bugreport.cgi?bug=511912

Note that the current version of the Xorg driver in unstable has issues
with xrandr, use the version in testing (squeeze).

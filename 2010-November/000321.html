I believe I've found a solution. It's a bit of a hack but it seems to work 
well enough.

I modified the dejitter module from this:

     ORIGINAL CODE:
        if (djt->nr == 1)
                samp [count] = *s;

    NEW CODE:
       if (djt->nr  ==  1){
                        samp [count].tv = s->tv;
                        samp [count].x = s->x;
                        samp [count].y = s->y;
                        samp [count].pressure = 0;
                    }

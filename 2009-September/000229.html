Hello,
I am trying to run tslib on a powerpc based board ADS5121 with a touchscreen
ELO 1715L. I built kernel with ELO Touch Serial Driver support. I cross
compiled tslib succesfully and put tslib files on my board.

A folder is created  "./sys/bus/serio/drivers/elo" with contents "bind,
bind_mode,    description,  uevent,   unbind". And when I "cat ttyPSC0" I
get some output from touch screen.
My env parameters are:

export TSLIB_CONSOLEDEVICE=/dev/ttyPSC0
export TSLIB_CONFFILE=/etc/ts.conf
export TSLIB_CALIBFILE=/etc/pointercal
export TSLIB_FBDEVICE=/dev/fb0
export TSLIB_TSDEVICE=/sys/bus/serio/drivers/elo/uevent
export TSLIB_TSEVENTTYPE=INPUT
export TSLIB_PLUGINDIR=/lib/ts


in ts.conf file I uncomment "module_raw input" .
Everything is ok till here. The problem starts now.
When i run ts_calibrate, I get :

Couldnt load module input
No raw modules loaded.
ts_config: Success

Do I make a mistake in the configuration? Actually I am suspicious about the
value of TSLIB_TSDEVICE. But there is no "/dev/input/event" kind of thing.
What should i do?

Thanks for the input. I can now run the ts_calibrate program to generate
the tslib pointercal file for use with Qt.



Just FYI, all that was needed in order to do this was to add one line to
aura.c:



set_bit(EV_KEY, aura.input_dev->evbit);



And then replace the occurrences of ABS_X / Y in the tslib input-raw.c file
with ABS_MT_POSITION_X / Y.

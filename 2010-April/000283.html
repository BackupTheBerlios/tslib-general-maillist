Hi Mike,

Can you check whether your touchscreen is sending events properly.

Send me the capture of evtest with input event interface device of the
touch screen. The tslib requires a x,y,press and sync parameters to be
reported from linux interface. If the parameters are not reported
properly tslib will not respond.

Also you can compile tslib enabling debug option for more information
on your problem.

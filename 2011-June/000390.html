Hi,

I am using ARCHOS 101 internet tablet.
I have ported QT on Armstrong as well as compiled tslib for ARCHOS 101
internet tablet.

Added a new file (/etc/ts.conf) with the following lines:

# Uncomment if you wish to use the linux input layer event interface
#module_raw input

# Uncomment if you're using a Sharp Zaurus SL-5500/SL-5000d
#module_raw collie

# Uncomment if you're using a Sharp Zaurus SL-C700/C750/C760/C860
# module_raw corgi

# Uncomment if you're using a device with a UCB1200/1300/1400 TS interface
#module_raw ucb1x00

# Uncomment if you're using an HP iPaq h3600 or similar
#module_raw h3600

# Uncomment if you're using a Hitachi Webpad
# module_raw mk712

# Uncomment if you're using an IBM Arctic II
module_raw arctic2

module pthres pmin=1
module variance delta=30
module dejitter delta=100
module linear

I have also setup the following environment variables for tslib:

export TSLIB_CALIBFILE='/etc/pointercal'
export TSLIB_CONFFILE='/etc/ts.conf'
export TSLIB_FBDEVICE='/dev/fb0'
export TSLIB_PLUGINDIR='/usr/lib/ts'
export
TSLIB_TSDEVICE="/dev/input/by-path/platform-ohci-omap.0-usb-0:2:1.0-event"
export
QWS_MOUSE_PROTO="Linuxinput:/dev/input/by-path/platform-ohci-omap.0-usb-0:2:1.0-event"

Now when I run the ts_calibrate utility which is provided with the tslib
package to calibrate the touchscreen, the crosshair appears in top left

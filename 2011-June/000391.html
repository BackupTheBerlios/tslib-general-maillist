> Now when I run the ts_calibrate utility which is provided with the tslib

> After calibration the touchscreen still isn't accurate.
> I would like to know whether I am missing anything, if so please let me

Unless I'm missing something, you appear to be using an input layer
event device, yet you're using the 'IBM Artic II' module_raw module?
Is this really correct for your hardware? We do have module_raw
input.. In addition, if the calibration is failing, it's almost
certainly getting useless touchscreen events from the device you
specified. Run ts_print_raw to see what it receives when you touch the
touchscreen. These are the uncalibrated values which ts_calibrate is
operating from.

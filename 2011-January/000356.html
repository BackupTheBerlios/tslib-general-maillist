On Fri, Jan 14, 2011 at 1:00 PM, David Rea <dave at daverea.com> wrote:
> Here are the commands necessary to cross-compile tslib for the iMX51:
>
> `git clone https://github.com/kergoth/tslib.git
>
> && cd tslib`
> `./autogen.sh`
> `./configure --prefix="/path/to/install-result"
> CC=/path/to/arm-linux-gcc --build=i386-linux --host=arm-linux
> --target=arm`
> `make && make install`
>
> From there, the binaries in tslib/install-result/ must be added to the
> target rootfs. Once the rootfs is downloaded, set the following
> environment variables...
>
> TSLIB_DEVICE=/dev/input/event0
> TSLIB_PLUGINDIR=/lib/ts
> TSLIB_CONFFILE=/etc/ts.conf
> TSLIB_CALIBFILE=/etc/pointercal
> QWS_MOUSE_PROTO=tslib:/dev/input/event0 <-- optional; connects Qt's
> QWS with tslib

FYI, if you use a more standard prefix (e.g. --prefix=/usr), and
instead run make DESTDIR=path/to/install-result install to get the
files where you want them, tslib should be able to find pointercal,
ts.conf, etc just fine using the default paths, so you could stick to
just setting TSLIB_DEVICE and QWS_MOUSE_PROTO.

> module.

> TSLIB_TSDEVICE=/dev/input/event13

> module_raw input
> module_raw galax
> module pthres pmin=1
> module variance delta=30
> module dejitter delta=100

First, you should only ever use one module_raw module. Either use input or
use galax. When using /dev/input/event*, the input module_raw module should
work fine, as far as I'm aware.

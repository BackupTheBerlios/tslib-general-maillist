Just adding to the last reply: the configure script has also the
options -L and -I that allows you to specify the location of headers
and libraries for the cross compiled tslib you are using.

There is also the option:

-qt3support

which is enabled by default (QT 4.6.0). Maybe it would be simpler for
you changing to QT4 but keeping your application using QT3 style.

> It works with the multi-touch input event device. (/dev/event?)

> multi-touch input device protocol.

>
> you should first download the orginal tslib, and replace(you may want
> to diff) the
> configure.ac(int tslib dir) and Makefile.am(int tslib/plugin dir), and
> then add mtinput-raw.c to plugin dir.
>
> after make and install, the plugin module named mtinput.
> in the config file,
> use :
> module_raw mtinput
> instead of :
> module_raw input
>

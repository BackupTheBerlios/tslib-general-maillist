hello, as i know ,the command ./configure will invoke a shell script (named "configure" ,just in your current file).if no configure file exist,then no configure can be make. 
so you just need a "configure" file,as in other softwares' way.
but in tslib's way,you need to run "./autogen.sh" to generate the configure file,(note both are shell scripts).

you can refer to the INSTALL file to see the details. and learn more by reading the helpfiles in the    like document.
wish it may help.
